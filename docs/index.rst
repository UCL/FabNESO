Welcome to FabNESO's documentation!
===================================

`Neptune Exploratory SOftware (NESO) <https://github.com/ExCALIBUR-NEPTUNE/NESO>`_
plugin for `FabSim3 <https://github.com/djgroen/FabSim3>`_, facilitating execution of
NESO simulations on both local and remote high performance computing systems via a
unified interface.
